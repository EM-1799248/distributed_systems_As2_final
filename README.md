# Instructions to run
- requires Gson jar

1. Compile
   ``` Bash
   make
   ```

2. Start the Aggregation Server
   ``` Bash
   make run-server
   ```

3. Start the Content Server
   ``` Bash
   make run-content
   ```

4. Send a GET Request
   ``` Bash
   make run-client
   ```

5. Clean Compiled Files
   ``` Bash
   make clean
   ```
